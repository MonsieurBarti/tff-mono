## Agent skills

### Issue tracker

Issues live as GitHub issues in this repo. Use `gh` CLI for all operations. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles mapped to GitHub labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context monorepo. `CONTEXT-MAP.md` at the root points to per-context `CONTEXT.md` files in `apps/tff-cc/`, `apps/tff-pi/`, and `packages/core/`. System-wide ADRs live in `docs/adr/`. See `docs/agents/domain.md`.
